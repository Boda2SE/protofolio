Cybersecurity Graduation Project

Description

This project is a Cybersecurity Web Application developed as a university graduation project. It focuses on simulating common cyberattacks and teaching users how to defend against them. The platform includes modules for password security, phishing detection, and system hardening. I personally developed around 75% of the system, including the core back-end logic and interface.

Key Features

  • Simulating common attacks like SQL Injection, XSS, and Brute Force.
  • Real-time threat monitoring dashboard.
  • User training and phishing simulation tests.
  • Admin interface for tracking user progress and logs.
  • Authentication system with two-factor login and secure session handling.
