CV Analysis AI Project

Description

This project is an AI-powered CV Analysis system developed as part of a university graduation project. It enables automated evaluation and comparison of submitted CVs against specific job requirements. The system uses Natural Language Processing (NLP) and machine learning algorithms to extract key information, assess qualifications, and rank candidates. I personally developed approximately 75% of the overall functionality.

Project Features

  • Upload and parse resumes in PDF format.
  • Extract skills, education, and experience using AI.
  • Automatically match CVs to job descriptions.
  • Rank candidates based on relevance and qualification.
  • Admin panel to view results and accept/reject applicants.
  • Notification system for accepted candidates.
